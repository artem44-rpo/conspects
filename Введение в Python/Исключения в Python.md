это объекты, сигнализирующие об ошибках, позволяющие обрабатывать их грациозно без остановки программы. Основная конструкция — try-except для перехвата ошибок. Все исключения наследуются от BaseException, базовый тип — Exception.

## Конструкция try-except

Блок try содержит код, который может вызвать исключение; except перехватывает его по типу. Синтаксис: try: ... except ValueError as e: print(e). Можно использовать except Exception для всех или except: для любого.

## Блоки else и finally

Блок else выполняется, если в try не было исключения. Finally всегда выполняется в конце, даже при ошибке или return — идеально для cleanup (закрытие файлов). Полный синтаксис: try: ... except: ... else: ... finally: ...

## Оператор raise

Raise принудительно вызывает исключение: raise ValueError("Ошибка"). В except можно raise для повторной передачи вверх: raise. С аргументами: raise Exception("Сообщение") from другая_ошибка.

## Собственные исключения

Создайте класс, наследующий Exception: class MyError(Exception): pass. В __init__ можно добавить детали: super().__init__(msg). Используйте raise MyError("Описание") для кастомных ошибок.
