Условия в Python реализуются с помощью конструкции if, которая позволяет выполнять код в зависимости от истинности выражения. Если условие истинно (True), выполняется блок кода под if, иначе код пропускается. Отступы (обычно 4 пробела) определяют начало и конец блока.

## Оператор if-else

Оператор else добавляется после if для выполнения альтернативного блока кода при ложном условии. Синтаксис: if условие: ... else: ... Это создаёт логическое ветвление для двух вариантов.

## Оператор elif

Для нескольких условий используется elif (else if), который проверяется последовательно после ложного if или предыдущего elif. Синтаксис: if условие1: ... elif условие2: ... else: ... Может быть любое количество elif.

## Логические и сравнения

Условия строятся из операторов сравнения: == (равно), != (не равно), >, <, >=, <=. Логические операторы: and (оба истинны), or (хотя бы одно истинно), not (инверсия). Приоритет: not > and > or.

## Truthy и Falsy

В Python не только True/False оцениваются в условиях: falsy — None, 0, пустые коллекции ([] , {} , ""), ложные строки; остальные — truthy. Функция bool() проверяет это: bool("") == False. Это упрощает проверки на пустоту

## Вложенные условия

Условия можно вкладывать: if внутри другого if для сложной логики. Пример:

`if x > 0:     if x < 10:        print("Маленькое")`

Избегайте глубокого вложения — используйте elif или and.

## Тернарный оператор

Для выражения вместо блока: значение1 if условие else значение2. Пример: status = "Да" if age >= 18 else "Нет". Упрощает короткие if-else.
