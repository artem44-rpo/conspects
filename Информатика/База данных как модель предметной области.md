База данных (бд) - это совокупность данных организованных по определённым правилам, отражающее состояние объекта и отношений в некоторой конкретной области, предназначена для хранения данных во внешней памяти или для постоянного применения. Базу данных можно рассматривать как информационную модель предметной области. 
Предметная область это целевая тема конкретного проекта. В свою очередь доменная (или концептуальная) модель призвана выразить значение терминов и понятий.
### Типы баз данных
1. Реляционная БД (табличные), потому что можно представить данные в виде таблиц
2. Сетевые - в них может быть несколько взаимосвязей между таблицами (и несколько предков)
3. Иерархические - модель, организующая данные в древовидную структуру, где каждый дочерний элемент подчиняется только одному родительскому

Подводя итог, можно сказать, что базы данных являются фундаментальным инструментом для упорядочивания и хранения данных. Классификация баз данных на реляционные, сетевые и иерархические демонстрирует разнообразие способов организации связей между данными, где каждый тип оптимизирован под определенную логику взаимодействия элементов (таблицы, графы или деревья).
