# Конспект по протоколу удалённого подключения RDP

## Что такое RDP

**RDP (Remote Desktop Protocol)** — это протокол удалённого доступа, разработанный компанией Microsoft. Он позволяет подключаться к другому компьютеру через сеть и полностью управлять им так, как будто пользователь находится непосредственно перед этим компьютером.

Основные возможности RDP:

- удалённое управление рабочим столом;
- запуск программ;
- работа с файлами;
- использование буфера обмена;
- передача принтеров, дисков и других устройств.

---

# Требования для подключения

Для успешного подключения необходимо:

- компьютер должен быть включён;
- на компьютере должен быть разрешён удалённый рабочий стол;
- знать IP-адрес или имя компьютера;
- иметь логин и пароль пользователя;
- компьютер должен быть доступен по сети.

---

# Подключение через встроенную утилиту mstsc

В Windows имеется стандартная программа **mstsc.exe** (Remote Desktop Connection), которая используется для подключения по протоколу RDP.

## Шаг 1

Нажать сочетание клавиш:

```
Win + R
```

---

## Шаг 2

Ввести команду:

```
mstsc
```

и нажать **Enter**.

---

## Шаг 3

В открывшемся окне ввести:

- IP-адрес компьютера;
- либо имя компьютера.

Например:

```
192.168.1.15
```

или

```
SERVER-01
```

Нажать кнопку **Подключить**.

---

## Шаг 4

При появлении окна авторизации ввести:

- имя пользователя;
- пароль.

При необходимости подтвердить сертификат безопасности.

---

## Шаг 5

После успешной авторизации откроется удалённый рабочий стол, через который можно работать с компьютером.

---

# Дополнительные настройки

Перед подключением можно открыть раздел **Показать параметры**, где доступны настройки:

- разрешение экрана;
- качество изображения;
- локальные принтеры;
- буфер обмена;
- локальные диски;
- звук;
- скорость соединения.

---

# Преимущества RDP

- высокая скорость работы;
- встроенная поддержка Windows;
- безопасная передача данных при правильной настройке;
- возможность удалённого администрирования;
- удобная работа с сервером или рабочим компьютером.

---

# Недостатки

- требуется стабильное интернет-соединение;
- необходима правильная настройка безопасности;
- открытый RDP-порт может стать целью злоумышленников;
- при работе через интернет рекомендуется использовать VPN.

---

# Вывод

Протокол **RDP** является стандартным средством удалённого управления компьютерами под управлением Windows. Для подключения используется встроенная утилита **mstsc**, которая позволяет быстро подключиться к удалённому компьютеру по IP-адресу или имени устройства, введя учётные данные пользователя.
