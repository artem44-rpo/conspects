# Функции в Python

## Что такое функция

Функция — это блок кода, который выполняет определенную задачу и может вызываться многократно.

## Создание функции

```python
def hello():
    print("Hello")
```

## Аргументы

```python
def greet(name):
    print(f"Hello, {name}")
```

## Возврат значения

```python
def add(a, b):
    return a + b
```

## Зачем нужны функции

- Повторное использование кода
- Удобство поддержки
- Разделение логики программы

## Что я запомнил

Функции помогают писать более чистый и понятный код.
