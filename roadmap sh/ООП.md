# Объектно-ориентированное программирование

## Основные понятия

- Класс
- Объект
- Атрибут
- Метод

## Пример

```python
class User:
    def __init__(self, name):
        self.name = name

    def say_hello(self):
        print(f"Привет, {self.name}")
```

## Создание объекта

```python
user = User("Артем")
user.say_hello()
```

## Преимущества ООП

- Удобная структура проекта
- Повторное использование кода
- Масштабируемость

## Что я запомнил

Класс — шаблон, объект — экземпляр класса.
